# EaglercraftX Bookmarklet
## Instructions
1. Copy this code:
**```javascript:(function(){var s=document.createElement('script');s.src='https://cdn.jsdelivr.net/gh/th3nu8/eagler@latest/bookmark.js';document.body.appendChild(s);})();```**
3. Make a bookmark with it
4. Play EaglercraftX!
## Make sure popups are allowed!!!
